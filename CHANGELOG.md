## [0.1.0] - 2025-02-09

### Added
- Initial release of diff_json library
- Basic JSON diff functionality
- Support for object comparison (added, removed, modified fields)
- Support for array comparison (with order preservation option)
- Support for nested JSON structures
- Multiple output formats (plain text, compact, JSON)
- Comprehensive test suite
- Documentation and examples
